# RemindPassword

여러 사이트 계정의 비밀번호를 자주 잊어먹는 사용자를 목적으로 처음 제작해 본 앱으로,<br/>
Remind Password는 사용자의 개인 정보를 암호화된 데이터베이스에 안전하게 보관할 수 있습니다.

## Features

* Base64를 사용하는 데이터 암호화
* 비밀번호 및 생체 인식 인증
* 즐겨찾기 기능
* 간편한 계정 및 데이터 삭제
